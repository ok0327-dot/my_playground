# 경제 블로그 자동화 프로젝트 지침서

## 프로젝트 개요

네이버 경제 블로그 운영을 AI 자동화로 지원하는 프로젝트.
기존 네이버 블로그(애드포스트 승인 완료)를 활용하여 경제/투자 전문 콘텐츠를 효율적으로 생산하고, 동일 콘텐츠를 개인 웹사이트에도 발행하여 이중 수익화한다.

---

## 운영자 프로필

- **본업**: 감사/회계 + 에너지 수입 분야 실무자 (평일 8시간 근무)
- **전문 영역**: 에너지/원자재, 재무 분석, 경제 지표 해석
- **기술 역량**: AI 자동화 파이프라인 구축 (Python, GitHub Actions, API 연동, Vercel 배포)
- **언어**: 한국어/영어 이중 언어
- **기존 자산**: 네이버 블로그 (애드포스트 승인 완료), 블로그 글 모음 PDF 보유
- **투입 가능 시간**: 하루 10~15분 (아침 출근 전 검토/발행)

## 사업 구조

### 채널 구조

```
[네이버 블로그] ← 한국 트래픽 (신뢰도 높음)
├── 애드포스트 수익
├── 쿠팡파트너스 (관련 도서, 맥락 있는 경우만)
└── CPA 제휴 링크 (텐핑, 링크프라이스)

[개인 웹사이트 (Next.js + Vercel)] ← 구글 트래픽
├── 구글 애드센스 수익 (금융 카테고리 고단가)
├── 블로그 글 약간 변형하여 게재
└── 향후 웹앱(투자 테스트 등) 추가 가능
```

### 수익 구조

```
Layer 1: 광고
├── 네이버 애드포스트 (블로그)
└── 구글 애드센스 (웹사이트) ← 금융 키워드 고단가, 최우선 승인 목표

Layer 2: 제휴 마케팅
├── 쿠팡파트너스 (투자 서적 등, 맥락 있는 경우만)
└── CPA 플랫폼 (텐핑, 링크프라이스 - 증권사 앱 설치, 보험 상담 등)

Layer 3 (향후): 웹앱 + 유료 콘텐츠
├── 투자 리스크 진단 테스트
├── 원자재 수혜 종목 계산기
└── 프리미엄 분석 리포트
```

## 핵심 자동화 파이프라인

### 구조

```
[매일 자동 실행 - GitHub Actions]

Step 1: 데이터 수집
├── 네이버 급상승 검색어 (경제 카테고리 필터링)
├── 네이버 경제 뉴스 인기 기사 TOP 10 (제목+링크만)
├── 구글 트렌드 급상승 키워드
└── 주요 수치 보충 (환율, 유가, 코스피 등)

Step 2: AI 분류
├── 일회성 이슈 → 패스
├── 반복 구조형 이슈 → 채택
└── 장기 분석 가능 주제 → 우선 채택

Step 3: AI 초안 생성 (Groq API)
├── 글 템플릿 구조 적용
├── 운영자 문체/톤 반영 (PDF에서 추출)
└── 전문가 관점 해설 포함

Step 4: 구글 시트에 저장
├── 제목, 본문, 키워드, 추천 태그
├── 관련 쿠팡/CPA 링크 (해당 시만)
└── 발행 여부 체크박스

[매일 수동 - 5~10분]
Step 5: 운영자 검토 → 네이버 블로그 수동 발행

[주 1~2회 추가 - 5분]
Step 6: 네이버 글 변형 → 웹사이트 발행
```

### 절대 금지 사항

- ❌ Selenium 등으로 네이버 블로그 자동 발행 금지 (계정 정지 리스크)
- ❌ 뉴스 본문 복사/요약 금지 (저작권 + 네이버 저품질 판정)
- ❌ 동일 시간대/동일 구조 반복 발행 금지 (자동화 패턴 감지됨)
- ❌ 매번 쿠팡파트너스 링크 삽입 금지 (체류시간↓, 신뢰↓)

## 콘텐츠 전략

### 타겟 독자

- **누구**: 투자에 관심 있는 30~40대 직장인
- **상태**: 주식/ETF 좀 해봤지만 체계가 없다
- **니즈**: "내가 잘하고 있는 건가?" 불안
- **톤**: 전문적이지만 쉽게, 재미있게

### 글 유형 (우선순위 순)

**1순위: 반복 구조형 분석 글 (장기 자산)**
- "금리 동결이 환율에 미치는 구조적 영향"
- "유가 상승기에 주목할 에너지 ETF 분석"
- "재무제표로 보는 위험한 기업 신호 5가지"
- → 검색 유입이 지속되는 에버그린 콘텐츠

**2순위: 고단가 SEO 타겟 글 (애드센스 승인용)**
- "ETF 초보 가이드: 2026년 추천 ETF 비교"
- "ISA 계좌 vs 연금저축: 직장인 절세 전략"
- "CMA 통장 금리 비교"
- → 금융 키워드는 애드센스 CPC 최상위

**3순위: 재미형 콘텐츠 (바이럴/트래픽)**
- "1년 전 오늘 삼성전자 샀다면 수익률은?"
- "커피값으로 매일 ETF 샀다면 지금 얼마?"
- → 공유 유도, 신규 방문자 유입

**4순위: 전문성 차별화 글 (경쟁 우위)**
- 에너지/원자재 시장 분석 (본업 전문성)
- 감사/재무 관점의 기업 분석
- → 다른 블로거가 쓸 수 없는 콘텐츠

### 글 템플릿 구조

```
[제목]: 검색 키워드 포함, 호기심 유발

[핵심 변화]: 무슨 일이 있었나 (2~3줄)
[왜 반복되는가]: 구조적 원인 분석
[과거 사례]: 비슷한 상황에서 어떤 결과가 있었나
[현재 수치]: 관련 데이터 (환율, 지수, 유가 등)
[투자자 체크 포인트]: 실질적 조언

[관련 콘텐츠 링크] (내부 링크)
[관련 도서/상품] (쿠팡파트너스, 맥락 있는 경우만)
```

### 콘텐츠 방향 원칙

- 뉴스 요약이 아닌 **구조적 분석**에 집중
- 일회성 이슈보다 **반복되는 메가 키워드** (금리, 환율, 유가, 배당, ETF, 연준)
- 운영자의 에너지/감사 전문성을 자연스럽게 녹여낼 것
- 초보 투자자(30~40대 직장인)가 이해할 수 있는 수준으로 작성

## 기술 스택

### 자동화 파이프라인
- **스케줄링**: GitHub Actions (매일 아침 자동 실행)
- **수집/가공**: Python (requests, BeautifulSoup)
- **AI 글 생성**: Groq API (빠른 처리, 비용 거의 0) 또는 Claude API
- **저장**: Google Sheets API (초안 저장 + 검토용)
- **발행**: 수동 (네이버 블로그), 반수동 (웹사이트)

### 개인 웹사이트
- **프레임워크**: Next.js (App Router, SSR - SEO 최적화)
- **호스팅**: Vercel (무료 티어)
- **DB**: Supabase (무료 티어, 필요시)
- **분석**: Google Analytics 4
- **광고**: 구글 애드센스

### 운영 비용
- 도메인: 연 ~15,000원
- Vercel/Supabase: 무료
- AI API (Groq): 거의 무료
- 총: **월 ~10,000원 이하**

## 실행 로드맵

### Stage 1: 기반 구축 (1~2개월)
- [ ] 기존 블로그 글 PDF 분석 → 문체/톤/주제 패턴 추출
- [ ] 자동화 파이프라인 MVP 구축 (수집 → 분류 → 초안 → 시트 저장)
- [ ] 네이버 블로그에 경제 분석 글 20개 발행
- [ ] 개인 웹사이트 구축 (Next.js + Vercel, 심플한 블로그 구조)
- [ ] 구글 애드센스 승인 신청 (고단가 금융 키워드 글 중심)
- **KPI**: 월 방문자 3,000 / 애드센스 승인 완료

### Stage 2: 확장 (2~4개월)
- [ ] 콘텐츠 자동화 안정적 운영 확인
- [ ] CPA 플랫폼 연동 (텐핑, 링크프라이스)
- [ ] 이메일 구독 수집 시작
- [ ] 웹앱 1개 추가 검토 (투자 리스크 진단 테스트)
- **KPI**: 월 방문자 10,000 / 월 수익 20~50만원

### Stage 3: 수익 최적화 (4~6개월)
- [ ] CPA 전환 최적화 (A/B 테스트)
- [ ] 시즌 웹앱/콘텐츠 추가
- [ ] 유료 PDF 리포트 테스트
- [ ] SNS 운영 여부 데이터 기반 판단
- **KPI**: 월 방문자 50,000 / 월 수익 100만원+

## 현재 최우선 작업

1. **기존 블로그 글 PDF 분석** → 문체, 톤, 주제 패턴 추출 → AI 프롬프트에 반영
2. **자동화 파이프라인 MVP** → 데이터 수집 + AI 분류 + 초안 생성 + 구글 시트 저장
3. **개인 웹사이트 기본 세팅** → Next.js + Vercel 초기 구조

## 핵심 원칙

1. **작게 시작, 데이터로 판단, 확신 있으면 확장**
2. **자동 생성은 하되, 자동 발행은 하지 않는다**
3. **뉴스 요약이 아닌 구조적 분석 글을 쓴다**
4. **본업 병행 가능한 범위(하루 10~15분)를 절대 초과하지 않는다**
5. **초기엔 수익보다 체류시간/공유율/재방문율을 추적한다**
6. **한국 트래픽은 네이버 블로그, 글로벌 트래픽은 웹사이트로 분리한다**
